# Security & Compliance - Learning Resources

Curated resources for learning application security, network security, cryptography, compliance, and security engineering.

## YouTube Channels & Tutorials

| Topic | Channel/Video | Level | Why Watch It |
|-------|--------------|-------|--------------|

| Cryptography | Computerphile - Cryptography Playlist | Intermediate | University researchers explaining encryption, hashing, and TLS |
| OWASP Top 10 | OWASP Foundation - Official Videos | Beginner | Direct from the organization that defines web security standards |
| Penetration Testing | John Hammond - CTF & Security | Intermediate | Capture-the-flag walkthroughs and security tool tutorials |

| Incident Response | SANS Institute - IR Talks | Advanced | Real-world incident response techniques from security professionals |

## Recommended Books

| Topic | Book | Author | Level | Why Read It |
|-------|------|--------|-------|-------------|
| Security Engineering | "Security Engineering" | Ross Anderson | Advanced | The comprehensive guide to building secure systems — free online |
| Web Security | "The Web Application Hacker's Handbook" | Dafydd Stuttard & Marcus Pinto | Intermediate | Systematic approach to finding and fixing web vulnerabilities |
| Cryptography | "Serious Cryptography" | Jean-Philippe Aumasson | Intermediate | Modern cryptography explained without heavy math |
| Secure Design | "Threat Modeling" | Adam Shostack | Intermediate | How to think about security during system design — STRIDE framework |
| DevSecOps | "Agile Application Security" | Laura Bell et al. | Intermediate | Integrating security into agile development workflows |
| Network Security | "Network Security Assessment" | Chris McNab | Advanced | Practical network penetration testing techniques |
| Cloud Security | "Hacking APIs" | Corey Ball | Intermediate | API-specific security testing and vulnerability patterns |
| Privacy | "Privacy Engineering" | Nishant Bhajaria | Intermediate | Building privacy-respecting systems at scale |
| Incident Response | "Incident Management for Operations" | Rob Schnepp et al. | Intermediate | Structured approach to handling security incidents |
