# Development Methodologies - Learning Resources

Curated resources for learning Agile, Scrum, DevOps, CI/CD, testing strategies, and development workflows.

## YouTube Channels & Tutorials

| Topic | Channel/Video | Level | Why Watch It |
|-------|--------------|-------|--------------|
| Agile Basics | Atlassian - "What is Agile?" | Beginner | Clear introduction to Agile principles from the makers of Jira |
| Scrum Framework | Mountain Goat Software - Scrum Series | Beginner | Mike Cohn's practical Scrum explanations with real team examples |
| DevOps Principles | TechWorld with Nana - DevOps Bootcamp | Beginner | End-to-end DevOps overview from CI/CD to monitoring |
| CI/CD Pipelines | Fireship - "CI/CD in 100 Seconds" | Beginner | Rapid overview of continuous integration and deployment concepts |
| Test-Driven Development | Continuous Delivery - TDD Series | Intermediate | Dave Farley's practical TDD approach with refactoring examples |
| Git Workflows | The Coding Train - Git and GitHub Series | Beginner | Visual, friendly introduction to version control |
| Code Review | Google Engineering Practices | Intermediate | How Google approaches code review — standards and best practices |
| Kanban | Atlassian - Kanban Explained | Beginner | Simple overview of Kanban boards, WIP limits, and flow metrics |
| Software Testing | Fireship - "Testing Explained" | Beginner | Quick overview of unit, integration, and end-to-end testing |
| Extreme Programming | Martin Fowler - XP Talks | Advanced | Foundational ideas behind pair programming, TDD, and continuous integration |

## Recommended Books

| Topic | Book | Author | Level | Why Read It |
|-------|------|--------|-------|-------------|
| Agile | "Agile Estimating and Planning" | Mike Cohn | Beginner | Practical approach to planning sprints, releases, and user stories |
| DevOps | "The Phoenix Project" | Gene Kim, Kevin Behr, George Spafford | Beginner | A novel that teaches DevOps principles through a compelling story |
| CI/CD | "Continuous Delivery" | Jez Humble & David Farley | Intermediate | The definitive guide to build, test, and deployment automation |
| Testing | "Test Driven Development: By Example" | Kent Beck | Intermediate | The original TDD book — learn by building a money example step by step |
| Lean | "The Lean Startup" | Eric Ries | Beginner | Apply lean methodology to product development and validation |
| Team Dynamics | "Accelerate" | Nicole Forsgren, Jez Humble, Gene Kim | Intermediate | Data-driven analysis of what makes high-performing engineering teams |
| Software Delivery | "The DevOps Handbook" | Gene Kim et al. | Intermediate | Practical companion to "The Phoenix Project" with real implementation guides |
| Refactoring Process | "Working Effectively with Legacy Code" | Michael Feathers | Advanced | Techniques for safely improving code that has no tests |
