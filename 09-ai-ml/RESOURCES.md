## Recommended Books

| Topic | Book | Author | Level | Why Read It |
|-------|------|--------|-------|-------------|
| ML Fundamentals | "Hands-On Machine Learning with Scikit-Learn, Keras, and TensorFlow" | Aurelien Geron | Beginner | The best practical ML book — covers end to end with code examples |
| Deep Learning | "Deep Learning" | Ian Goodfellow, Yoshua Bengio, Aaron Courville | Advanced | The foundational deep learning textbook — theory and math |
| Statistics for ML | "An Introduction to Statistical Learning" (ISLR) | James, Witten, Hastie, Tibshirani | Intermediate | Statistical foundations for ML with R examples — free online |
| NLP | "Speech and Language Processing" | Dan Jurafsky & James Martin | Advanced | Comprehensive NLP textbook covering classical and modern methods |
| MLOps | "Designing Machine Learning Systems" | Chip Huyen | Intermediate | Production ML systems — data management, deployment, and monitoring |
| Data Engineering | "Fundamentals of Data Engineering" | Joe Reis & Matt Housley | Intermediate | The data engineering lifecycle from ingestion to serving |
| Reinforcement Learning | "Reinforcement Learning: An Introduction" | Sutton & Barto | Advanced | The definitive RL textbook — free online |
| AI Ethics | "Weapons of Math Destruction" | Cathy O'Neil | Beginner | How algorithms can cause harm — essential for responsible AI |
| Feature Engineering | "Feature Engineering for Machine Learning" | Alice Zheng & Amanda Casari | Intermediate | Practical techniques for transforming raw data into ML features |
