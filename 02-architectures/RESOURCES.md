# System Architecture - Learning Resources

Curated resources for learning system design, architectural patterns, microservices, event-driven architecture, and distributed systems.

## YouTube Channels & Tutorials

| Topic | Channel/Video | Level | Why Watch It |
|-------|--------------|-------|--------------|
| System Design Basics | ByteByteGo - System Design Series | Beginner | Visual diagrams explaining real-world architectures (Netflix, Uber, etc.) |
| Microservices | TechWorld with Nana - Microservices Explained | Beginner | Clear overview of microservice patterns with Kubernetes context |
| Distributed Systems | Martin Kleppmann - Distributed Systems Lectures | Advanced | University-level lectures from the author of "Designing Data-Intensive Applications" |
| Event-Driven Architecture | Confluent - Kafka & Event Streaming | Intermediate | Deep dives into event sourcing, CQRS, and stream processing |
| API Design | Fireship - RESTful APIs Explained | Beginner | Quick, visual overview of REST, GraphQL, and gRPC |
| Database Architecture | CMU Database Group - Andy Pavlo Lectures | Advanced | University lectures on database internals and query optimization |
| Design Patterns | Gaurav Sen - System Design Playlist | Intermediate | Interview-focused system design with clear trade-off discussions |
| Scalability | InfoQ - Architecture Conference Talks | Advanced | Real talks from engineers at Google, Netflix, Amazon about production systems |
| CQRS & Event Sourcing | Greg Young - CQRS/ES Talks | Advanced | The original creator explaining the pattern in depth |
| Message Queues | Hussein Nasser - Message Queues Playlist | Intermediate | Practical comparison of Kafka, RabbitMQ, SQS with real use cases |

## Recommended Books

| Topic | Book | Author | Level | Why Read It |
|-------|------|--------|-------|-------------|
| Distributed Systems | "Designing Data-Intensive Applications" | Martin Kleppmann | Advanced | The modern bible of distributed systems — every senior engineer should read this |
| System Design | "System Design Interview" (Vol 1 & 2) | Alex Xu | Intermediate | Step-by-step walkthroughs of designing 20+ real systems |
| Architecture Patterns | "Fundamentals of Software Architecture" | Mark Richards & Neal Ford | Intermediate | Comprehensive overview of architecture styles and trade-offs |
| Microservices | "Building Microservices" | Sam Newman | Intermediate | The definitive guide to microservice design and decomposition |
| Event-Driven | "Building Event-Driven Microservices" | Adam Bellemare | Advanced | Patterns for event sourcing, CQRS, and stream processing |
| Domain-Driven Design | "Implementing Domain-Driven Design" | Vaughn Vernon | Advanced | Practical DDD with bounded contexts and aggregates |
| API Design | "Designing Web APIs" | Brenda Jin, Saurabh Sahni, Amir Shevat | Intermediate | Best practices for RESTful API design and versioning |
| Scalability | "The Art of Scalability" | Abbott & Fisher | Advanced | Scale cube framework and organizational scaling patterns |
| Database Internals | "Database Internals" | Alex Petrov | Advanced | How databases work under the hood — B-trees, LSM-trees, replication |
